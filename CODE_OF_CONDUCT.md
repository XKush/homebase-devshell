# Contributor Covenant Code of Conduct

## Our pledge

We pledge to make participation in the **DevReady / HomeBase DevShell** project a harassment-free experience for everyone.

## Expected behavior

- Be respectful in issues and pull requests  
- Focus on constructive feedback  
- Accept that OSS users have different skill levels  

## Unacceptable behavior

- Harassment, trolling, or personal attacks  
- Publishing others' private information  

## Enforcement

Report concerns via [GitHub Security advisories](https://github.com/XKush/homebase-devshell/security/advisories) or open a private contact with the maintainer.

This project follows the spirit of the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.
