# Справочник команд (RU)

> **Продукт:** [README.ru.md](../../README.ru.md) · **English:** [README.md](../../README.md)

Навигация по command center после установки HomeBase DevShell.

## Навигация

| Команда | Что делает |
|---------|------------|
| `go` / **Ctrl+Alt+G** | Двухуровневое меню: **[следующий]** из home → категории → Enter=действие |
| `menu` / `palette` | То же, что `go` |
| `sec` / **Ctrl+Alt+S** | Только Tor + PGP |
| **Ctrl+Alt+K** | `komandy` — справочник |
| **Ctrl+Alt+B** | `home` — обзор |

**Enter** = выполнить · **Esc** = назад · **Ctrl+/** = справка

## Быстрый старт (после install)

1. `devshell doctor` — окружение готово  
2. `home` — обзор  
3. `go` — меню действий
