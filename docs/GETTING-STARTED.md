# Getting started

1. **Install** (copy from [README](../README.md))
2. **Close and reopen** your terminal
3. **Run doctor** — look for `Ready to work`

```powershell
pwsh -File $HOME\.homebase\devshell\devshell.ps1 doctor
```

That's the whole flow.
